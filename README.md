# The StatTrak Bot

### A simple StatTrak bot for all of your tracking needs

## Functionality

- All inputs will be via Discord text chat

- Note all numbers have to be positive

```txt
-Create a Tracker
>create [TRACKER NAME]

-Add to a tracker
>add [TRACKER] [NUMBER]

-Remove from a tracker
>remove [TRACKER] [NUMBER]

-Delete a tracker
>delete [TRACKER]

-Show a single tracker
>show one [TRACKER]

-Show all trackers
>show all
```
